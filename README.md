Web 3 cli for blockchain interaction and utilities


# Gen bin

yarn package:bin

https://github.com/vercel/pkg

## create bin archive

```sh
tar -cvzf ./bin-dist/w3t-linux.tar.gz ./bin/w3t-linux

sha256sum ./bin-dist/w3t-linux.tar.gz

```
